# EmoLab Extended Abstract

This directory contains an IEEE-format extended abstract describing the EmoLab system and its demonstration workflow.

## Document

**[EmoLab: Synchronized Multimodal Affective Analysis and Replay](EmoLab_Demo_Extended_Abstract.pdf)**

The extended abstract presents:

* the motivation for synchronized multimodal affect-related inspection;
* the shared temporal representation used by EmoLab;
* deterministic replay of preserved source media and model outputs;
* transcript, speaker, face, prosody, vocal-burst, and language inspection;
* cross-modal comparison;
* EmoiTextGraph;
* the affective-cognitive interpretation layer;
* the demonstration protocol;
* ethical considerations and limitations.

## Publication Status

This document is provided as a reusable project and demonstration paper.

It has not been presented as a formally accepted or peer-reviewed publication.
